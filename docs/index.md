## 想动手改文档？

可以，走正规流程。

> 🛠️ PR 提交教程   
> 
> 1.	打开仓库地址
> 
> 2.	点你想编辑的 .md 文件（具体文件名可以查看链接），点右上角的「📝 铅笔」图标
> 
> 3.	修改内容后，页面底下写个提交说明，比如 “修正错别字”
> 
> 4.	选择「Create a new branch」（默认选项就行）
> 
> 5.	点「Propose changes」→「Create pull request」

⸻

PR 发过来后我们会有人审。格式用Markdown写。
